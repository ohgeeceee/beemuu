# Welcome

This is the very first thread in the BeeEmUu community forum.
It's a test post to make sure the index generator works and to
give newcomers a place to land when they arrive.

## How this works

- Threads are markdown files in `community/forum/threads/`.
- You file one by opening a pull request.
- Replies are appended to the same file under a `## Replies`
  heading. The git log is the timeline.
- Search this repo's `community/forum/` folder via your editor or
  `git grep "P0420" community/forum/threads/`.

## What this is *not*

- It's not a real-time chat.
- It's not an upvote-driven ranking site.
- It's not a place to paste configs — link to them by path with a
  sha256 checksum.
