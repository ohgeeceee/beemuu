Community profiles

TOML profiles under community/profiles/ that show up in the hosted panel's preset list. Reviewers can disable a profile without deleting the file.

Coming in PR 4

Read-only render of each .toml file with a "disable / enable" toggle and a "view raw" link.

API contract (PR 4): GET /admin/api/community/profiles, GET /admin/api/community/profiles/<slug>, POST /admin/api/community/profiles/<slug>/toggle.
