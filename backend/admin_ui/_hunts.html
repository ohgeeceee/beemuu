Hunt challenges

Bug-bounty / gamified hunt challenges. Each challenge has a slug, a title, a point value, and a payload blob that ships to clients.

Coming in PR 4

Create / edit / enable / disable challenges. Soft-delete by setting enabled = 0; the row stays so historical leaderboard entries still resolve.

API contract (PR 4): GET /admin/api/hunts, POST /admin/api/hunts, POST /admin/api/hunts/<id>, POST /admin/api/hunts/<id>/toggle.
